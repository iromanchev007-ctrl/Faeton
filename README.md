# Фаэтон — сайт автосервиса с онлайн-записью

Современный адаптивный лендинг автосервиса «Фаэтон» (Саратов). Клиент
оставляет марку, модель и симптомы — заявка моментально приходит менеджеру
в Telegram. Без звонков.

## Состав

```
index.html              — разметка страницы (Hero, О компании, Услуги,
                          Заявка, Адрес, Связь, Подвал)
styles.css              — стили, адаптив, анимации появления
script.js               — меню, анимации, марки/модели авто, отправка формы
api/lead.js             — серверная функция отправки в Telegram (Vercel)
netlify/functions/lead.js — то же для Netlify
netlify.toml            — конфиг Netlify (+ редирект /api/lead)
vercel.json             — конфиг Vercel (cleanUrls, заголовки безопасности)
```

## Куда вставить ключ бота и chat_id

Токен **не вписывается в код сайта** — он живёт в переменных окружения
хостинга, поэтому не попадёт к посетителям.

Нужны две переменные:

| Переменная           | Где взять                                            |
|----------------------|------------------------------------------------------|
| `TELEGRAM_BOT_TOKEN` | у [@BotFather](https://t.me/BotFather) при создании бота |
| `TELEGRAM_CHAT_ID`   | у [@userinfobot](https://t.me/userinfobot) или [@getmyid_bot](https://t.me/getmyid_bot) |

### Vercel
1. Импортируйте репозиторий на [vercel.com](https://vercel.com) (Framework Preset: **Other**).
2. **Settings → Environment Variables** → добавьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`.
3. **Deploy** (или Redeploy, если переменные добавили позже).

### Netlify
1. Импортируйте репозиторий на [netlify.com](https://netlify.com).
2. **Site settings → Environment variables** → добавьте те же две переменные.
3. Деплой. `netlify.toml` уже настроит функцию и редирект `/api/lead`.

> Быстрый тест без переменных: можно временно вписать значения в блок
> `FALLBACK` в начале `api/lead.js` (или `netlify/functions/lead.js`).
> **Не коммитьте токен в публичный репозиторий** — он утечёт.

## Бот: получаем chat_id за минуту
1. Создайте бота у @BotFather → получите токен.
2. Напишите своему боту любое сообщение (например, `/start`).
3. Узнайте свой ID у @userinfobot — это и есть `TELEGRAM_CHAT_ID`.
   Для группы добавьте бота в группу и используйте ID группы (начинается с `-`).

## Локальный просмотр
Откройте `index.html` в браузере. Форма отправки заработает только на
хостинге с настроенной функцией (на Vercel/Netlify) или через
`vercel dev` / `netlify dev`.
